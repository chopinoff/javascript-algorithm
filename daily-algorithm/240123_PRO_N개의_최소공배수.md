## 문제 해석

-   두 수의 배수 중 공통이 되는 가장 작은 숫자 찾기

## 최종 제출 코드

-   `arr`에서 숫자 두 개를 뽑아 최소 공배수를 구하는 것을 반복했습니다.
-   최소 공배수를 구하기 위해 재귀 함수(`getGcd`)를 사용했습니다.
    -   a, b의 최소 공배수 = a \* b / 최대 공약수

```js
function getGcd(a, b) {
    if (b === 0) return a;
    return getGcd(b, a % b);
}

function solution(arr) {
    return arr.reduce((a, b) => (a * b) / getGcd(a, b));
}
```
